# ZX-Spider-48K-LBA
ZX Spider 48K Amended by liveboxandy to use 9 Pin MiniDIN and move resistors from under IC's.

Original files from https://github.com/konkotgit/ZX-48-Spider and redrawn in Kicad 10 using the gerbers from konkotgit as an initial layout.
Autorouting along with some manual actions were used for the final pcb layout and gerber generation.

See https://github.com/konkotgit/ZX-48-Spider for more details and any issues that might be seen.
